# Data Availability

The benchmark uses the public Dataset of 3D Garments with Sewing Patterns:

https://zenodo.org/records/5267549

The raw archives are not redistributed here. The manuscript reports the verified compressed corpus size as 84.77 GB. Experiment outputs and summary artifacts derived from the local verification run are included under `experiments/`.

To reproduce the full pipeline, download the Zenodo archives into:

```text
raw/zenodo_3d_garments_sewing_patterns/archives/
```

Then run the scripts listed in `README.md`.
